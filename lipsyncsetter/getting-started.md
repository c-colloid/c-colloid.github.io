# 導入方法

## インストール

### VCCでインストール（推奨）

1. VRChat Creator Companion (VCC) を開く
2. **Settings** → **Packages** → **Add Repository**
3. 以下のURLを入力:
   ```
   https://c-colloid.github.io/vpm/index.json
   ```
4. 「Add」をクリック
5. プロジェクトを開き、「Manage Project」から **LipSyncSetter** を追加

## 起動方法

```
Tools → LipSyncSetter
```
