# FXCreator

> FXレイヤーの作成を補助するUnity拡張

## Overview

FXCreatorは、VRChatアバターのFXレイヤー作成を効率化するツールです。

::: warning 準備中
詳細ドキュメントは準備中です
:::

## Quick Start

1. **VCCでインストール**
   - リポジトリURL: `https://c-colloid.github.io/vpm/index.json`

2. **ツールを起動**
   - メニュー: `Tools` → `FXCreator`

詳しくは[導入方法](./getting-started)をご覧ください。
