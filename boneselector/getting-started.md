# 導入方法

## インストール

### VCCでインストール（推奨）

1. VRChat Creator Companion (VCC) を開く
2. **Settings** → **Packages** → **Add Repository**
3. 以下のURLを入力:
   ```
   https://c-colloid.github.io/vpm/index.json
   ```
4. 「Add」をクリック
5. プロジェクトを開き、「Manage Project」から **BoneSelector** を追加

### unitypackageでインストール

1. [GitHub Releases](https://github.com/c-colloid/BoneSelector/releases) からunitypackageをダウンロード
2. Unityプロジェクトにドラッグ&ドロップ

## 起動方法

BoneSelectorはエディターウィンドウとして動作します。以下の3つの方法で起動できます：

### 方法1: メニューバーから

```
Tools → BoneSelector
```

### 方法2: Hierarchyの右クリックメニュー

Hierarchy上で右クリック → **BoneSelector** を選択

### 方法3: ツールバーから

ツールバーのコントロールメニュー（一番右端のメニュー）から **BoneSelector** を選択

## ボーン表示の開始

1. エディターウィンドウの **Avatar** 欄にアバターをセット
2. **Button** を押すとボーンの表示が開始されます

::: tip
VRC Avatar Descriptorが付いたアバターのみが対象です
:::
