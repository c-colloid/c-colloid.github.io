# BoneSelector

> シーン上でアバターのボーンを可視化・選択するUnity拡張

## Overview

BoneSelectorは、Unityエディタ上でVRChatアバターのボーン構造を直感的に確認・操作するためのツールです。

## Features

| 機能 | 説明 |
|-----|------|
| **ボーン可視化** | アバターのArmature以下のオブジェクトをシーン上に表示 |
| **ワンクリック選択** | シーン上でボーンを直接クリックして選択 |
| **範囲選択** | ドラッグで複数ボーンを一括選択 |
| **表示カスタマイズ** | ボーン/ワイヤー/スフィアの3形式から選択可能 |
| **Playモード対応** | 再生中も動作し、揺れものの確認に使用可能 |

## Use Cases

**こんな時に便利：**
- PhysBoneの設定確認
- ボーン階層の把握
- 特定ボーンの素早い選択
- アニメーション・揺れものの動作確認

## Quick Start

1. **VCCでインストール**
   - リポジトリURL: `https://c-colloid.github.io/vpm/index.json`

2. **ツールを起動**
   - メニュー: `Tools` → `BoneSelector`

3. **アバターを選択してボーン表示**
   - エディタウィンドウでアバターをセット → ボタンを押す

詳しくは[導入方法](./getting-started)をご覧ください。
